# React-Todo-List

A simple and interactive To-Do List application built with **React**.  
This project helps users manage daily tasks by adding, completing, and deleting to-dos in a clean UI.

---

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and user-friendly interface
- Built with reusable React components
- You can also design the page on your own by using css

---

## 🛠️ Tech Stack

- **React**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **Node.js & npm**
